User Infomation Fusion

Introduction

Uif system aims to fuse user's infomation, which scattered in several social media. In other words, to complement single user's different information from multi social media, resulting complete user information.

Data Source

In this period, we determined to use Twitter and Facebook as sample social media, to fusing information of users who use these two SNS mutually.

Process

text -- tokenization -- pos tagging -- entity extraction -- relevance word extraction -- complementing

Go Start !

Result

The information will be presented as entities the users mentioned and the relevance words of those entities.
